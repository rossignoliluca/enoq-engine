- Is specific (right catalyst for right reaction)
- Is not consumed (can repeat)
- Withdraws when the process is complete

**Applied to humans:**
- The task wants to be done → ENOQ helps it complete

- The decision wants to be made → ENOQ helps it complete
- The threshold wants to be crossed → ENOQ helps it complete

## The Test

For any feature, design, or decision:

1. **Does this help something complete?** (If not, why are we doing it?)
2. **Is ENOQ acting as catalyst?** (Facilitating, not creating?)
3. **Will ENOQ withdraw after?** (Or does this create dependency?)
4. **Is this the right catalysis for this domain?** (Specific, not generic?)

If all four are yes, proceed.
If any is no, reconsider.
